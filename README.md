# 1-Hour Research Agent Challenge

## The Task

Build a Python script that answers user questions by performing live web searches and synthesizing the results using an LLM.

## The Stack

- **Search:** Tavily API
- **LLM:** Together AI `
- **Coding Agent:** Your choice (Cursor, Claude, etc.)
